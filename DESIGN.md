# DESIGN.md — DURDOM AI STUDIO

> Design-system контекст для AI-агента (Antigravity / Cursor / Claude Code).
> Формат по спецификации google-labs-code/design-md. Клади в корень проекта или вставляй в контекст перед версткой.

## 1. Product tone & principles
DURDOM AI STUDIO — AI-продакшн студия для **артистов, шоу и брендов**. Тон: кинематографичный, тёмный, дерзко-премиальный, серьёзный. Ощущение — «пост-продакшн студия, которая делает визуал для сцены и клипов», а не инфобизнес-лендинг.
Принципы:
- Тьма как холст, красный как единственный акцент. Контент (видео/кадры) — главный герой, интерфейс — тихая рама.
- Крупная дисплейная типографика, много воздуха, графичная строгость.
- Кинематографичность > декоративность. Каждый эффект работает на «дорого и серьёзно».
- Доказательства делом: работы, спеки, пайплайн. Ноль воды про «заработай миллион».

## 2. Colors (роли, hex, применение)
- **Void** `#08080a` — самый глубокий фон, подложка под слои, футер.
- **Base** `#0d0d0f` — основной фон страницы.
- **Surface** `#16161a` — карточки, панели, модалки.
- **Red** `#e11d2a` — ЕДИНСТВЕННЫЙ акцент: primary-кнопки, активные табы, цифры, hover-границы, глоу, тех-подписи.
- **Red Deep** `#8f1319` — заливки под красным, hover-состояния, глубина, градиентные хвосты.
- **Ink** `#f2f2f4` — заголовки и основной текст.
- **Muted** `#a8a8b0` — вторичный текст, подписи, дескрипторы.
- **Line** `#2a2a30` — границы карточек, разделители, хэйрлайны.
Правило: красным закрашено ≤10% площади экрана. Никаких вторых акцентных цветов. Никаких синих/фиолетовых «AI-градиентов».

## 3. Typography
- **Display** — `Unbounded` (700–800), UPPERCASE, tracking плотный/минусовой на крупных кеглях. Hero 72–96px, line-height 0.96. Секционные H2 40–56px.
- **Body** — `Inter` (400–600), 16–18px, line-height 1.6, цвет Muted для абзацев, Ink для акцентов.
- **Mono / tech-label** — `JetBrains Mono` (500), 12–13px, UPPERCASE, letter-spacing 0.2em, цвет Red. Для надзаголовков секций (`01 · ПРОДАКШН`), тех-спеков, метаданных кейсов.
Иерархия строгая: display / h2 / h3 / body / mono-label. Ничего между.

## 4. Spacing
Базовая сетка 8px. Токены: 8 / 16 / 24 / 32 / 48 / 64 / 96 / 128.
- Внутренние отступы карточек: 24–32.
- Вертикальный ритм секций: 96–128 между блоками (desktop), 56–72 (mobile).
- Контейнер max-width 1200px, боковые поля 24 (mobile) / 48+ (desktop).

## 5. Components
- **Buttons** — капсульные (radius 100px). Primary: заливка Red, текст Base, hover → Red Deep + лёгкий glow. Secondary: прозрачная, border Line, hover → border Red + текст Ink. Высота 48–56, padding 20–28.
- **Cards** — фон Surface, border 1px Line, radius 14. Hover: border Red, `translateY(-4px)`, тень `0 0 28px rgba(225,29,42,.18)`. Media-карточки (работы) — border-radius 14, кадр во всю карточку, play-оверлей по центру, красная рамка на hover.
- **Nav** — sticky, фон Base с blur backdrop, border-bottom Line, при скролле добавляется тень. Логотип слева, пункты по центру/справа, primary-кнопка справа.
- **Tabs (фильтр портфолио)** — pill-табы, активный залит Red, неактивные — текст Muted с border Line.
- **Accordion (FAQ)** — строки-разделители Line, иконка +/− красная, плавное раскрытие.
- **Tech-spec chips** — маленькие mono-плашки (4K · Hap · 30fps · LED) с border Line, для кейсов и hero.
- **Section label** — mono-надзаголовок Red + номер (`03 · РАБОТЫ`).

## 6. Layout
- Одноколоночный поток с крупными секциями, чередование плотности: воздушный hero → плотная сетка работ → воздушный процесс.
- Сетки: услуги 3 колонки (desktop) / 1 (mobile); портфолио 2–3 колонки с крупным первым рядом; арсенал 2–3 колонки.
- Асимметрия допустима: крупный кадр + текстовый блок сбоку в кейсах.
- Хэйрлайны Line + короткий красный сегмент как «тех-разметка» на границах секций.

## 7. Motion
Движок: GSAP + ScrollTrigger + Lenis (инерционный скролл). Только transform/opacity, will-change, throttle.
- Hero-заголовок: mask-reveal по буквам (clip-path снизу), stagger 40ms, лёгкий cursor-параллакс.
- Reveal-on-scroll: заголовки split на слова, поднимаются из-под маски.
- Pinned horizontal showreel: секция «Работы» пинится, работы скроллятся горизонтально (scrub).
- Магнитные карточки + микро-tilt (3–5°) + красный glow на hover.
- Count-up метрик, бесконечный marquee тегов, clip-path вайпы между секциями, тонкий красный cursor-trail (off на touch).
Обязательно `prefers-reduced-motion` → анимации упрощаются/отключаются. На mobile тяжёлые эффекты (pin, cursor) отключить.
Тайминги: 400–600ms, ease `power3.out`. Дорого и плавно, не «казино».

## 8. Accessibility
- Контраст текста к фону ≥ 4.5:1 (Ink/Muted на Base проверить; Muted не опускать ниже #a8a8b0).
- Focus-состояния видимые (красная обводка 2px).
- Aria на аккордеоне, каруселе, модалке, табах. Клавиатурная навигация.
- Видео — без автозвука, с контролами; alt у изображений.
- Уважать reduced-motion.

## 9. Do-not-use (запреты)
- Никаких вторых акцентных цветов, синих/фиолетовых AI-градиентов, неоновой радуги.
- Никаких светлых фонов и «Tailwind-стартер» дефолтов (белые карточки, синие кнопки, чистый Inter-заголовок без характера).
- Никакого инфобизнес-глянца: 3D-стрелок, взрывов эмодзи, «жирных обещаний дохода» на первом экране.
- Не заливать красным крупные площади. Не мельчить дисплейный шрифт. Не смешивать больше трёх уровней типо-иерархии в одном блоке.
- 18+/OFM-контент НЕ выносить в общее портфолио и hero — только в закрытый блок «Арсенал» с плашкой.
